# flir

**0x01** --> 刷入Bootloader.zip内的hex;

**0x02** --> 使用串口连接电脑，使用SuperLink.exe连接;

**0x03** --> 按住key4开机进入Bootloader界面;

**0x04** --> 按照提示输入，使用Ymodem格式发送主程序的.bin文件。


*這裏是插值8x的工程（master為5x），實際體驗并沒有什麽卵用
